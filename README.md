
# NEXAGUARD

> **AI-Powered Token & Wallet Analyzer**  
> Akıllı kontratlar, wallet ve token'lar için otomatik analiz, risk sınıflandırması ve veri görselleştirme sistemi.  
> Full-stack blockchain projesi — Sui SDK, Express.js backend ve React frontend ile geliştirilmiştir.
Kripto ekosisteminde token güvenliğini analiz etmek için tasarlanmış yapay zekâ destekli bir güvenlik analizi platformudur.

Kullanıcının girdiği token / wallet adresi üzerinden zincir verilerini çeker, istatistiksel göstergeleri analiz eder ve “güvenli”, “riskli” veya “tehlikeli” gibi bir sınıflandırma sonucu üretir.  
Bu sistem hem geliştiriciler hem yatırımcılar için, on-chain istatistikleri anlamlandırarak riskleri görünür kılmayı amaçlar.
